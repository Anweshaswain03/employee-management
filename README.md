# Employee Management System

## Project Overview
The Employee Management System is a Java-based console application that manages employee records using Collections Framework (ArrayList and HashMap). The application supports CRUD operations, employee search, salary reporting, file handling, and exception handling.

## Objectives
- Store employee information efficiently.
- Implement CRUD operations.
- Search employees by ID.
- Generate salary reports.
- Save and load employee data from files.
- Handle exceptions gracefully.

## Technologies Used
- Java
- ArrayList
- HashMap
- File Handling
- Exception Handling
- Scanner Class

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/employee-management-system.git
   ```

2. Navigate to the project directory:
   ```bash
   cd employee-management-system
   ```

3. Compile the program:
   ```bash
   javac EmployeeManagementSystem.java
   ```

4. Run the program:
   ```bash
   java EmployeeManagementSystem
   ```
